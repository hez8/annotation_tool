# 图像标注工具

用于图像的 Bounding Box 标注桌面工具。

## 功能

- **图片浏览**：打开本地文件夹，左右按钮或方向键快速切换图片
- **缩放标注**：支持放大/缩小/适应窗口/原始大小，滚轮缩放，便于标注细小物体
- **12 类别**：按钮式类别选择，点击后在图像上拖拽画框
- **框微调**：通过左/上/右/下四个参数独立调节标注框，支持步长配置
- **标注管理**：「下一个」存储当前标注继续标注，「确定」保存全部标注
- **JSON 结构体存储**：标注结果存为同名 `.txt`，采用 `{}` 分级嵌套的 JSON 格式
- **自动加载**：打开图片时自动搜索同名 txt 并导入已有标注
- **自动保存**：切换图片时自动保存未存储的标注

## 环境依赖

- Python 3.7+
- Pillow（PIL）

```powershell
conda activate py39
pip install Pillow
```

## 启动方式

```powershell
conda activate py39
cd "标注目录"
python fod_annotation_tool.py
```

## 使用流程

```
打开文件夹 → 自动加载第一张图片
    ↓
[点击类别按钮] 选择标注类别（如"金属"）
    ↓
[鼠标拖拽] 在图像上画框 → 红色虚线预览
    ↓
[微调] 通过左/上/右/下按钮或手动输入调整框位置
    ↓
[点击「下一个」] 标注存入结果列表 → 可继续选择其他类别标注
    ↓
... 重复以上步骤标注所有目标 ...
    ↓
[点击「确定 ✓」] 保存到同名 .txt 文件
    ↓
[◀▶] 切换图片，继续下一张
```

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| `←` / `→` | 切换上一张/下一张图片 |
| `Ctrl` + `+` / `-` | 放大 / 缩小 |
| `Ctrl` + `0` | 原始大小（100%） |
| `Esc` | 撤销当前标注框 |
| 鼠标滚轮 | 缩放图像 |
| `Ctrl` + `O` | 打开文件夹 |

## 标注文件格式

标注结果保存为与图片同名的 `.txt` 文件，内容为 JSON 结构体数组：

```json
[
  {
    "id": 0,
    "name": "金属",
    "bbox": {
      "x1": 120,
      "y1": 230,
      "x2": 315,
      "y2": 460
    }
  },
  {
    "id": 3,
    "name": "木头",
    "bbox": {
      "x1": 500,
      "y1": 320,
      "x2": 680,
      "y2": 510
    }
  }
]
```

| 字段 | 说明 |
|------|------|
| `id` | 类别编号（0-11） |
| `name` | 类别中文名称 |
| `bbox.x1` | 框左边界（像素） |
| `bbox.y1` | 框上边界（像素） |
| `bbox.x2` | 框右边界（像素） |
| `bbox.y2` | 框下边界（像素） |

## 修改标注

- **双击标注结果列表**中的某一行，可将该标注加载到画布上进行修改（框会重新显示为红色预览状态），修改后点击「下一个」重新存储
- 选中列表中的标注行，点击「删除选中」可删除该条标注

## 自定义保存目录

点击右侧「选择...」按钮可指定标注文件保存到其他目录。默认保存到图片所在目录。
